Instructions to run this C# program

Download this model into finer_model/onnx.model file: 
https://drive.google.com/file/d/14hk6x766VE_Rgy8jqm75DytfxUhxnJ87/view?usp=sharing

Then run:

```
dotnet build
dotnet run
```
